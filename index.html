Pseudoclases

Estilos especiales dependiendo del contexto, posicion o estado.

:root

La pseudo-clase de CSS :root selecciona el elemento raiz de un documento. Aplicando a HTML, :root representa el elemento <html> y es identico al selector html, excepto que su nivel de especificidad es mayor. Sirve para declarar variables CSS.

:hover

La pseudo-clase de CSS :hover se presenta cuando el usuario coloca el puntero sobre algún elemento, pero no necesariamente está activo. En pantallas touch :hover es problemático.

:active

La pseudo-clase :active de Css cubre el caso en el que un elemento está siendo activado por el usuario. Cuando se interactúa con el ratón, se corresponde al momento en el que el usuario presiona el botón del ratón hasta el momento en el que lo suelta. A menudo es usado en los elementos <a> y <button> de HTML.

El orden si importa:

  1. :link {...}
  2. :visited {...}
  3. :hover {...}
  4. :active {...}

:target

La pseudo-clase :target representa el elemento único, si existe alguno, con un id coicidente con el identificador de fragmentos de la URI del documento.

:nth-child :first-child :last-child

  • nth-child:selecciona el elemento indicado que sea enésimo hijo de su elemento padre.
  • first-child: selecciona el elemento indicado que sea el primer hijo de su elemento padre.
  • last-child: selecciona el elemento indicado que sea el último hijo de su elemento padre.

:nth-of-type :first-of-type :last-of-type

  • nth-of-type: selecciona el elemento indicando que sea el enésimo hijo en su tipo sin importar la jerarquía dentro del elemento padre.
  • first-of-type: selecciona el elemento indicado que sea el primer hijo en su tipo sin importar la jerarquía dentro del elemento padre.
  • last-of-type: selecciona el elemento indicado que sea el último hijo en su tipo sin importar la jerarquía dentro del elemento padre.

:not

La pseudo-clase CSS de negación, :not(X), es una notación funcional que toma un selector simple X como argumento. Coincide con un elemento que no está representado por el argumento. X no debe contener otro selector de negación.

:empty

La pseudo-clase :empty corresponde a un elemento sin ningún nodo hijo. Sólo cuando hay nodos de elemento o texto con uno o más caracteres dentro del elemento, ése se convierte en vacío.
